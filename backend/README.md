# Backend

This project uses **Supabase** as a Backend-as-a-Service (BaaS). 

Currently, the React frontend interacts directly with the Supabase API for:
- Authentication
- Database operations (PostgreSQL)
- Storage (Images)
- Realtime subscriptions

This folder is reserved for future backend expansions such as:
- Supabase Edge Functions
- Custom Node.js microservices
- Database migration scripts and schema definitions
